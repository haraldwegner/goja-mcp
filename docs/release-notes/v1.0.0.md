# goja-mcp v1.0.0

**GOJA** — compiler-accurate Java intelligence for AI agents. First public release, and the
start of a fresh version line.

## What this is

goja-mcp is the renamed, repackaged, AGPL-licensed successor to **javalens-mcp v1.12.0**. The
engine's capability is unchanged — Eclipse-JDT–backed analysis, navigation, and
behaviour-preserving refactoring over MCP, with auto-applying refactorings + one-call undo,
modernisation, null-safety, naming/Javadoc analysis, and duplicate-code detection — now under
the GOJA brand.

## Rebrand

- Java packages `org.javalens.*` → `org.goja.*`; runtime jar `javalens.jar` → `goja.jar`.
- MCP server identity → **GOJA**; environment variables `JAVALENS_*` → `GOJA_*`.
- Version line reset to **v1.0.0** — a clean break, not a continuation of the javalens v1.x line.
- Pairs with **[goja-studio](https://github.com/haraldwegner/goja-studio)** (the renamed
  javalens-manager desktop control plane).

Existing javalens-manager users: install **goja-studio**, which migrates your workspaces and
settings on first launch and downloads the goja-mcp runtime automatically.
