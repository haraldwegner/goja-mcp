# goja-mcp v2.0.0 — The local knowledge store (Sprint 21, the "learn" capstone)

GOJA remembers what worked *here*. v2.0 adds a **local, embedded, workspace-scoped knowledge store**
with a **denoising, terminal, grounded** retrieval contract — a cue resolves to **one** fitting node
(its pointer resolved to current code by the compiler) or an authoritative **absence**, never a
similarity-ranked pile. This fills the `Advisor` seam scaffolded in Sprint 18, so the orchestration
loop now reads *and* writes the store. **One net new tool** — the `experience` front door (surface
40 → 41). Ships paired with **goja-studio v2.0.0** (the push channel that injects recall).

## The store — embedded, workspace-scoped, H2-backed

- Opened at start, closed at stop, under the workspace `-data` dir (in-memory when none). No daemon,
  no network, works offline. `ExperienceStore` is a **port**; `H2ExperienceStore` is one adapter —
  a future networked service (Sprint 28) is a swap behind the same seam, not a rewrite.
- Entries reuse the shared `SymbolFact` shape + retrieval facets (status / scope / operation /
  alias-normalized symptoms / typed links / fault ownership).

## `experience(kind=…)` — one parametric front door

- **record** — store an observation as a `candidate` (type, summary, confidence, scope, symptoms,
  links, …).
- **recall** — TERMINAL retrieval for a cue (symbol / package / operation / symptom / external_system).
  **Two phases**: fuzzy keyword/alias gather → deterministic **scope-containment fit gate** → one
  fitting node (or a small closed, in-scope set) with the pointer JDT-resolved to current code, **or**
  an authoritative absence. Never a pile. `format=text` returns flat injection-ready lines.
- **primer** — the always-on **domain layer** (bounded contexts, concepts, invariants) for a
  session-start orientation.
- **load** — seed from memory files (`*.md`): frontmatter → type/summary, `symbol` → JDT-resolved
  pointer (stale flagged on ingest), `[[wikilinks]]` → edges; idempotent per source.
- **refresh** — re-resolve every pointer through JDT; a symbol that no longer exists is flagged
  `superseded` (dropped from recall). **wipe** / **promote** round out maintenance.

## The filled `Advisor` seam + write admission

`ExperienceAdvisor` replaces `NoOpAdvisor` at the orchestration wiring:

- **adviseBefore** → the terminal recall (≤1 fitting node), advisory only.
- **record(Outcome)** → the **write-admission filter**: admit only what the compiler can't re-derive —
  `ROLLED_BACK` → `failure_mode`, `FLAGGED` → `hazard`; a successful `APPLIED` (derivable) is dropped;
  duplicates deduped. Every write a `candidate` pending promotion.

## Anti-RAG, by contract

RAG returns the top-*k* similar chunks and makes the model sift them — it *grows* context. GOJA's
contract is the inverse: fit-gated + terminal, **resolve or dismiss**, always *shrinking* the problem.
Absence is a first-class, storable result. Embeddings (a later sprint) only ever gather Phase-1
candidates *behind* the fit gate — vectors never get the last word; scope-containment does.
